# 🎯 项目核心目标

## 一句话描述
**AI 嗓音女性化教练系统** — 通过渐进式唱歌+对话训练，锻炼喉部相关肌肉群（环甲肌、喉上提肌群等），重塑神经协调模式，最终让用户 **在现实中无需任何设备，自然发出女性化声音**。

---

## 核心原理

```
渐进式唱歌训练 → 环甲肌(CT肌)力量↑ + 神经协调模式重塑 → 日常对话自然调用 → 伪音达成
```

### 关键肌肉群
| 肌肉 | 作用 | 训练目标 |
|------|------|---------|
| 环甲肌（CT肌） | 拉长声带 → 音高↑ | 增强力量，成为主导发力肌 |
| 甲杓肌（TA肌） | 缩短声带 → 音高↓ | 抑制习惯性收紧 |
| 喉上提肌群 | 喉结上抬 → 共鸣腔缩小 | 持续激活 |
| 软腭提肌 | 打开鼻咽通道 → 头腔共鸣 | 协同训练 |

### 公式
```
肌肉力量 × 神经协调模式 = 自然发声效果
```

---

## 产品形态
一个 **AI 驱动的实时声音教练 App**（Web/移动端），包含：

1. **实时音高检测 + 可视化曲线**
2. **歌单分级训练模式**（Lv.1 男低 → Lv.5 女声）
3. **AI 对话教练模式**（跟读、自由对话、实时反馈）
4. **共鸣分析**（胸腔 vs 头腔能量比）
5. **自适应难度升降级**

---

## 训练分级体系

### 歌单难度阶梯
| 级别 | 音域范围 | 代表歌手 | 训练目标 |
|------|---------|---------|---------|
| Lv.1 男低 | C3-A3（130-220Hz） | 许嵩、毛不易 | 建立音高感知基线 |
| Lv.2 男中 | D3-C4（147-262Hz） | 周杰伦、陈奕迅 | 扩展低音区控制力 |
| Lv.3 男高 | F3-E4（175-330Hz） | 林俊杰、张信哲 | 激活CT肌，喉结上提 |
| Lv.4 中性 | A3-G4（220-392Hz） | 周深、苏打绿 | CT肌持续发力，共鸣迁移 |
| Lv.5 女声 | C4-C5（262-523Hz） | 邓紫棋、王菲 | 头腔共鸣主导，目标达成 |

### 对话难度阶梯
| 级别 | 目标音高 | 特征 |
|------|---------|------|
| D1 | 150-170Hz | 短句、慢速、单调 |
| D2 | 170-200Hz | 正常语速、带情绪 |
| D3 | 200-240Hz | 日常对话模拟、自然维持 |

## 系统架构概览

```
用户界面（实时音高曲线 + 共鸣热力图 + AI教练提示）
        │
   AI 教练引擎
   ├── 音高检测（F0提取）
   ├── 共鸣分析（频谱能量比）
   └── 难度自适应（自动升降级）
        │
   音频采集层
   ├── 麦克风输入 44.1kHz
   ├── 实时滑动窗口处理
   └── MFCC/F0/共振峰特征提取
```
